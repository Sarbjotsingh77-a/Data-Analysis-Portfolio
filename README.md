# 🚀 How to Run the Project

## Clone Repository

```bash

```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Launch Jupyter Notebook

```bash
jupyter notebook
```

Open any notebook from the `notebooks/` directory and run the cells.
